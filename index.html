Projetos

Site Android

Projeto de uma página web informativa sobre a história do mascote do Android. Desenvolvido com HTML5 e CSS3, o site possui estrutura semântica, layout responsivo, imagens adaptáveis, vídeo incorporado do YouTube, uso de variáveis CSS, fontes personalizadas e efeitos de interação no menu. O objetivo do projeto é praticar conceitos de estruturação de páginas, estilização e responsividade.

Acesse

Projeto Social

Projeto de interface que simula um smartphone exibindo páginas de redes sociais. Criado com HTML e CSS, utiliza um iframe para carregar diferentes telas (Home, GitHub, Instagram e LinkedIn) ao clicar nos ícones laterais. O projeto também inclui efeitos de hover, posicionamento com position e transform, e estilização com imagens e background responsivo.

Acesse

Projeto Login

Página de login responsiva desenvolvida com HTML5 e CSS3, contendo formulário de autenticação com campos de e-mail e senha, ícones do Material Icons e estilização com variáveis CSS. O layout se adapta a diferentes dispositivos utilizando media queries, alterando a disposição dos elementos entre mobile, tablet e desktop.

Acesse

Consulta de CEP

Aplicação web de consulta de CEP criada com HTML, CSS e JavaScript. O usuário digita o CEP e o sistema busca automaticamente os dados do endereço através da API ViaCEP, preenchendo os campos de rua, bairro e cidade. O projeto utiliza fetch API, async/await, manipulação do DOM e validação de formulário.

Acesse

Gerenciador de Tarefas

Este projeto é uma aplicação web de gerenciamento de tarefas desenvolvida com React e Vite. A aplicação permite ao usuário adicionar, remover, visualizar e organizar tarefas, ajudando na produtividade e no controle de atividades do dia a dia. O projeto foi criado com foco em praticar conceitos de componentes React, gerenciamento de estado com Hooks, manipulação de listas no React e armazenamento no localStorage, garantindo que as tarefas permaneçam salvas mesmo após recarregar a página.
